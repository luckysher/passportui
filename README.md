PasswordUI:
==========
Passport UI basically a Figma design to Frontend conversion project for the figma link:

https://www.figma.com/design/sQlrdPJpHrTDNfNgEdZZfk/Usman-Intern-Task--Copy-?node-id=354-886&t=L7lR5iuV2j98FYrx-0


Frameworks used:
----------------

    - JS Framework: React / Next Js
    - CSS Framework: Tailwind css
    - vite 
    - 100% Responsive


# How to install ?
download the project from the github repo and run the following npm command

    npm i or npm install

# How to run ?
        npm run dev


The frontend server will start on 5173 port number.
and you can checkout the frontend at the link localhost:5173 in your web browser.
